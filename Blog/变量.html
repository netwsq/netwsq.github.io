变量

请先看下面的代码:

#include<iostream>
using namespace std;
int main(){
int a,b;
cin>>a>>b;
cout<<a+b;
return 0;
}

上面的代码实现的是加法功能，在这段代码中，cin的作用是输入，并将两次输入的结果放到a和b中，这里可以复制到VS上自行实验。在C++中，储存数据的容器叫做变量，你可以赋值于它或者改变它的值，而一些我们不改变的变量叫做常量，例如我们可以在程序中定义一个π的值用来计算圆的大小，尽管我们自己可以改变它的数值，但是作为一个程序来说改变它的大小运行得到的就会与事实不符。
在C++中，每一个变量我们都要声明，也就是告诉程序，我创建了一个变量，这个变量里可以是数，也可以是字母等等，我们只需要用定义的函数加上给变量取的名称就可以创建一个变量，下面是部分定义函数的例子:

bool 存储值 true 或 false。
char 通常是一个字符（八位）。这是一个整数类型。
int 对机器而言，整数的最自然的大小。
float 单精度浮点值。1位符号，8位指数，23位小数。
double 双精度浮点值。1位符号，11位指数，52位小数。
void 表示类型的缺失。
wchar_t 宽字符类型。
C++ 也允许定义各种其他类型的变量，比如枚举、指针、数组、引用、数据结构、类等等，这将会在后续的章节中进行讲解。
以上函数这里不做过多阐述，若想了解自行搜索。

上面的代码中，int a,b;定义了两个分别叫做a和b的变量，cin使用户输入a和b的值，最终cout输出a+b的值。
当然，我们可以在变量后直接加=赋值于它例如:int a=10,b=20;
在C++中，=是一个赋值符号，它并不代表着相等，而是指使=前面的数变为=后面的数，例如:int a,b=10;a=b;cout<<a; 这个时候你会发现输出的是10。有的时候，我们想让a加上一个数x，我们可以这样理解，让a=a+x，这样可以实现加法，例如:

int a=10,b=20;
a=a+b;
cout<<a;

这个时候你会发现输出的是a+b也就是30. 但这样打出来好长啊，而且，一眼看上去也不能很快理解，所以，C++有一个语法，如果我们想让a加上x，我们可以这样:a+=x; 看，这样是不是简单多了呢?很多的时候，我们的变量都会加1，加1，不停的加1，例如后面要讲的循环。这个时候，a+=1;感觉也没那么简便了，于是，诞生了a++和++a,减法同理。但是，++a和a++的区别是什么呢?单独使用的时候是没有区别的，但是如果当成运算符，就会有区别了！我们用a=i++和a=++i举例说明
1.先说a=i++，这个运算的意思是先把i的值赋予a，然后在执行i=i+1；当i初始等于3时，执行a=i++，最终结果a=3，i=4
2.而a=++i，这个的意思是先执行i=i+1，然后在把i的值赋予a；当i初始等于3时，执行a=++i，最终结果a=4，i=4.
